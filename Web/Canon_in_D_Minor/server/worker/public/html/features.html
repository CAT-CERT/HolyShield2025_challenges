Rehearsal Flow

  • Ticket issue: Register for a rehearsal window and receive a pass.
  • Warmup lock: New tickets enter a short cooldown before admin cues.
  • Section split: Identical sections run in parallel to keep tempo true.
  • Cue testing: Signal checks verify internal routing before diagnostics.
  • Conductor run: Diagnostics request health checks and redirections.
