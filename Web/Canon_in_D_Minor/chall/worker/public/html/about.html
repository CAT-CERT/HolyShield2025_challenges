About the Rehearsal Grid

The rehearsal grid is built to keep a canon perfectly aligned. Each section runs its own warmups, while the conductor coordinates global tempo and alignment checks.

Tickets are issued for every rehearsal block. Sections must complete their warmups before the conductor requests diagnostics.

What makes it work

  • Section autonomy: Each part can warm up without stalling the ensemble.
  • Controlled cues: Conductor diagnostics only trigger when warmups complete.
  • Shared ticketing: Rehearsal passes sync across sections.
  • Local memory: Warmup state stays per section for precision.
